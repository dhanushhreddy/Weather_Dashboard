# 🌦️ Real-Time Weather Dashboard
A modern, responsive weather application built with HTML, CSS, and Vanilla JavaScript.

## Technical Highlights
* **Dual API Integration:** Uses Open-Meteo for weather data and Geocoding API for city-to-coordinate conversion.
* **Asynchronous Programming:** Implemented `async/await` for seamless data fetching without page reloads.
* **Dynamic UI:** Features a custom weather icon mapping system based on WMO weather codes.
* **Error Handling:** Built-in validation for empty inputs and "City Not Found" scenarios.
